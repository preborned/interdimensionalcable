# iCable
iCable (or Interdimensional Cable), brings you a simple and convenient solution to watch your own hosted media or external sources in an all-in-one, cable-like experience through your web browser.

(Currently only tested on Chromium 141.0.7390)
## Description
The iGuide is an interactive EPG that can read from any xmltv.xml file, and/or you can add your own custom "always live" channels. You can also add your own logo, connect an OpenWeather API to get an hourly forecast as well as the current temperature, connect a TMDb API to get posters for your content, customize all colors to your liking, add custom messages to the footer of the guide, and much more to come, all simply customizable from the config.xml file.

The On-Demand menu is still under construction and planning.

## Usability
iCable has only been tested on the latest Chromium (141.0.7390) and was built on Windows. iCable works in any Chrome or Chrome-based browser (such as Brave or MS Edge) on any PC operating system. This software is not designed to be used in mobile cases. As of this writing / today's updates, iCable is stable and working as-is but still lacks the complete development I'd like to do.
