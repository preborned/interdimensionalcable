# iCable
Originally Interdimensional Cable, iCable brings you a simple and convenient solution to watch your own hosted media or external sources in an all-in-one, cable-like experience.
## Description
iCable is an interactive guide that reads from any xmltv.xml file and allows you to add your own custom "always live" channels. You can also add your own logo, connect an OpenWeather API to get an hourly forecast as well as the current temperature, customize all colors to your liking, and much more to come, simply from the config.xml file.

If you are reading this, this code is in it's earliest stage. It isn't capable of much at all, and is still technically only a prototype. Stay tuned.
